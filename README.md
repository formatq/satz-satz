# satz-satz — German sentence modulator

One German sentence, four dials: **Subjekt · Verb · Zeitform · Genus Verbi**. Spin a dial, the sentence regenerates instantly, and the words that changed flash with a fading highlight — watch the article agree, the separable prefix jump to the end, the object become the subject in Passiv.

Sibling of [words-words](https://github.com/formatq/words-words) (English phrasal verbs), rebuilt N-dimensional. See `SPEC-DE.md` for the full design and `IMPLEMENTATION-NOTES.md` for the engineering notes.

## Controls

- `←` / `→` — move between dials, `↑` / `↓` — spin the active dial (hold to rattle)
- `1`–`4` — jump straight to a dial
- Mouse wheel over any dial spins it; click activates; touch drag works too

## Development

```sh
npm install
npm run dev        # local dev server
npm test           # navigation, generator, invariant + connectivity tests
npm run generate   # regenerate src/de/data/tuer.json from the tables
npm run build      # production build
```

All sentence variants are pre-generated by `scripts/generate-de.mjs` from small hand-encoded morphology tables and 8 declarative word-order frames, and committed as static JSON — zero network requests after page load. A validator (unit test + dev-mode startup check) enforces dataset invariants, including that every variant is reachable from the initial one by single-dial moves.

## Deploy

Pushes to `main` deploy to GitHub Pages via `.github/workflows/deploy.yml`. One-time setup: **Settings → Pages → Source: GitHub Actions** (the first run fails at `configure-pages` until this is set — re-run it after).
